# AlgoRhythm JS - Project Plan

## Vision
A clean, focused web app to study and master JavaScript algorithms and data structures.
Built from Colt Steele's "JS Algorithms and Data Structures Masterclass" course material,
enriched with extra tips, patterns, and interview-ready knowledge.

---

## Topics to Cover

### Foundations
- [x] Big O Notation
- [x] Big O of Arrays & Objects
- [x] Problem Solving Patterns (Frequency Counter, Multiple Pointers, Sliding Window, Divide & Conquer)
- [x] Recursion

### Searching & Sorting
- [x] Searching Algorithms (Linear Search, Binary Search, Naive String Search)
- [x] Elementary Sorting (Bubble Sort, Selection Sort, Insertion Sort)
- [x] Intermediate Sorting (Merge Sort, Quick Sort, Radix Sort)

### Data Structures
- [x] Singly Linked Lists
- [x] Doubly Linked Lists
- [x] Stacks & Queues
- [x] Trees & Tree Traversal (BST, BFS, DFS)
- [x] Binary Heaps & Priority Queues
- [x] Hash Tables
- [x] Graphs, Graph Traversal & Dijkstra's Algorithm

### Advanced
- [x] Dynamic Programming

---

## App Features

### Core
- [x] Sidebar navigation with all topics grouped by category
- [x] Home page with topic cards overview
- [x] Individual topic pages with explanations, code, and tips
- [x] Syntax-highlighted code blocks
- [x] Big O complexity tables
- [x] Tip/warning callout boxes
- [x] Clean, dark-mode design

### Each Topic Page Includes
- Clear, plain-English explanation of the concept
- Working JavaScript code examples with syntax highlighting
- Big O time & space complexity breakdown
- Key tips and common pitfalls
- Interview relevance notes
- Visual diagrams where helpful

## Tech Stack
- React 19 + Vite 7
- React Router v7 (client-side routing)
- Prism React Renderer (code syntax highlighting)
- Lucide React (icons)
- CSS (custom, no framework)

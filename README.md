# AlgoRhythm JS

Your go-to resource for mastering JavaScript algorithms and data structures.

Built from Colt Steele's "JS Algorithms and Data Structures Masterclass" course material, enriched with extra tips, real code examples, and interview-ready knowledge.

## What's Inside

**Foundations** - Big O Notation, Arrays & Objects performance, Problem Solving Patterns, Recursion

**Searching & Sorting** - Linear Search, Binary Search, Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Radix Sort

**Data Structures** - Linked Lists, Stacks, Queues, Trees (BST, BFS, DFS), Binary Heaps, Hash Tables, Graphs, Dijkstra's Algorithm

**Advanced** - Dynamic Programming (Memoization & Tabulation)

## Each Topic Covers

- Plain-English explanations (no jargon without context)
- Working JavaScript code with syntax highlighting
- Big O time & space complexity
- Tips, pitfalls, and interview notes
- Visual breakdowns where they help

## Running Locally

```bash
npm install
npm run dev
```
